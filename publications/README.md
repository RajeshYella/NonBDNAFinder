# Publications Directory

This directory contains publication-ready manuscripts based on the NonBDNAFinder codebase and validation results.

## Contents

### Main Publication

**NonBDNAFinder_Nature_Publication.md**

A comprehensive Nature-format scientific publication describing NonBDNAFinder, including:

- **Abstract**: Overview of the platform's capabilities and key findings
- **Introduction**: Background on non-B DNA structures and computational detection needs
- **Results**: 
  - Platform architecture and capabilities (11 classes, 24 subclasses)
  - Validation against NBST/Non-B DB tools (3.2× detection advantage)
  - Comparative genomics analysis (8 bacterial genomes, 133,434 motifs)
  - Disease-associated repeat expansion loci analysis (153 genes, 5,721 motifs)
  - Performance benchmarks
- **Discussion**: Significance, limitations, and future directions
- **Methods**: Detailed algorithm descriptions and validation methodology
- **Figures and Tables**: Publication-quality data presentations
- **References**: 27 peer-reviewed citations
- **Supplementary Information**: Extended methods and additional analyses
